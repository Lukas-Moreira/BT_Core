Client TCP Service

class BluPY.services.tcp_client.TcpClient

Classe responsável por realizar requisições HTTP para obter a data e hora atual de um servidor remoto. Methods

define_datetime(datetime_str)

Define a data e hora do sistema com base na string fornecida.

Parâmetros:

  • datetime_str (str): String representando a data e hora no formato ‘YYYY-MM-DD HH:MM:SS’.

get_datetime_now(host, port=9050)

Realiza uma requisição HTTP para obter a data e hora atual do servidor especificado. Parâmetros:

  • host (str): Endereço IP ou hostname do servidor.

  • port (int): Porta do servidor HTTP (padrão: 9050).
